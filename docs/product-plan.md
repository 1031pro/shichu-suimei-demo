# 商品プラン

## 基本方針

ココナラでは「四柱推命アプリを作ります」よりも、「占い師向けの四柱推命命式ツールを制作します」として出す。

理由は、購入者が欲しいのは一般ユーザー向け診断アプリではなく、自分の鑑定業務を効率化する専用ツールである可能性が高いため。

## ライトプラン

命式表示に必要な基本機能を提供する入口プラン。

含める機能:

- 生年月日・出生時刻入力
- 年柱・月柱・日柱・時柱の表示
- 天干・地支
- 蔵干
- 通変星
- 十二運
- 五行バランス
- 空亡
- 大運一覧
- 年運一覧
- 現在の大運・年運のハイライト
- スマホ対応
- 単独ページとして納品

ライトプランに含めないもの:

- 鑑定文生成
- PDF鑑定書
- 管理画面
- AI生成
- 複数流派の切り替え
- 複雑な独自判定
- 顧客データ保存

## スタンダードプラン

鑑定者の実務に合わせて表示項目や見た目を調整するプラン。

追加候補:

- 表示項目のカスタマイズ
- 鑑定者独自の項目追加
- 流派に合わせた一部ルール調整
- デザイン調整
- WordPress埋め込み
- ココナラ鑑定用の見やすい画面設計
- 印刷向けレイアウト

## プレミアムプラン

鑑定書・テンプレート・管理機能まで含める上位プラン。

追加候補:

- 鑑定文テンプレート
- 占い師独自の解釈文登録
- PDF鑑定書出力
- 顧客入力フォーム
- 顧客別の結果ページ
- 管理画面
- 複数流派の切り替え
- 独自ロジック実装
- AI鑑定文生成

## オプション設計

基本機能は命式を正しく見せるところまでに絞る。

鑑定文や鑑定書は、占い師ごとの価値が出る部分なので、オプションとして単価を上げやすい。

主なオプション:

- 鑑定文テンプレート追加
- PDF鑑定書出力
- 大運ごとの解釈文追加
- 年運ごとの解釈文追加
- 流派別ルール調整
- 独自項目追加
- 管理画面追加
- LINE・予約導線追加
- LP制作
- AI生成機能追加

## 訴求軸

- 無料サイトでは合わない流派・表示項目に対応
- スマホでいつでも命式・大運・年運を確認
- 鑑定者本人のスタイルに合わせて調整
- 鑑定文は勝手に作らず、先生の言葉を活かせる
- ココナラ鑑定、対面鑑定、講座資料作成に使える
