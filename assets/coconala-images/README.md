# ココナラ出品画像素材

ココナラ出品用の生成画像、デモスクリーンショット、合成後の完成画像を管理する。

## 命名

- `01-cover-base.png`: 表紙用の生成背景
- `01-cover.png`: 表紙
- `02-problem.png`: 悩み提起
- `03-solution.png`: 解決策
- `04-demo.png`: デモ画面
- `05-standard-features.png`: 標準機能
- `06-major-annual-luck.png`: 大運・年運
- `07-pwa-mobile.png`: PWA・スマホ対応
- `08-options.png`: オプション
- `09-precheck.png`: 購入前確認
- `10-consultation.png`: 見積相談導線
- `04-demo-screen.png`: デモ画面スクリーンショット
- `final-01-cover.png`: 文字入れ後の完成画像

## 注意

今回の `01` から `10` は image2.0 で文字入り生成した初版。
日本語文字は生成AIの性質上、細部が崩れることがあるため、最終出品前には目視確認する。
必要であれば、同じ構図を背景だけで再生成し、日本語文字を後から重ねる。
