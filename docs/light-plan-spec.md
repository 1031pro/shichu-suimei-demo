# ライトプラン確定仕様

## 位置づけ

ライトプランは、占い師向けに「スマホで命式・大運・年運を確認できる」最小商品。

鑑定文生成やPDF鑑定書は含めない。鑑定者本人が読むための命式表示ツールとして成立させる。

## 標準機能

### 入力

- 生年月日
- 性別
- 出生時刻

出生時刻は標準入力欄として用意する。ただし、時刻不要の流派にも対応できるように、設定で非表示・未使用にできる。

### 命式表示

- 年柱
- 月柱
- 日柱
- 時柱
- 天干
- 地支
- 蔵干
- 通変星
- 蔵干通変星
- 十二運
- 天中殺または空亡
- 五行バランス
- 陰陽バランス

時刻不要の設定では、時柱列を非表示または空欄にする。

### 運勢周期

- 大運一覧
- 大運の開始年齢
- 大運の干支
- 大運の通変星
- 大運の十二運
- 現在の大運ハイライト
- 年運一覧
- 現在年の年運ハイライト

大運・年運はライトプランに含める。

理由:

- 占い師の実務では命式だけだと不足しやすい
- 「今どの運気か」を確認できることが商品価値になる
- 追加鑑定文オプションへの導線になる

## ライトプランに含めないもの

- 鑑定文
- AI鑑定生成
- PDF鑑定書
- 顧客データ保存
- 管理画面
- 複数流派のUI切り替え
- 独自チャート
- 独自鑑定メソッドの資料化
- 真太陽時補正
- 早子時・夜子時の細かい指定
- WordPress埋め込み調整

これらはスタンダード以上、またはオプション扱い。

## 納品形式

初期は静的Webアプリを基本にする。

- `index.html`
- `assets`
- `js`
- `css`
- 設定JSON

サーバーやDBを使わず、ブラウザだけで動く構成を優先する。

理由:

- ココナラ納品が簡単
- 保守コストが低い
- 購入者のサーバー環境に依存しにくい
- スマホからもURLを開けば使える

## 販売時の説明文

ライトプランでは、四柱推命の鑑定に必要な基本情報をスマホ対応のWeb画面で自動表示します。

生年月日・性別・出生時刻を入力すると、命式、通変星、十二運、五行バランス、大運、年運などを確認できます。

鑑定文、PDF鑑定書、独自の流派調整、先生独自の鑑定資料化はオプションで対応します。

## 注意書き

四柱推命は流派や先生ごとに採用ルールが異なるため、標準仕様から外れるルール調整は追加費用の対象にする。

「完全な流派再現」をライトプラン内で保証しない。
