# 販売プラン設計

## 商品の見せ方

占いアプリ制作ではなく、占い師の仕事道具として見せる。

推奨表現:

> あなた専用の四柱推命命式ツールを制作します

避けたい表現:

> AIで自動鑑定します

理由:

占い師に刺さる価値は、自分の鑑定を置き換えることではなく、命式作成・運勢確認・資料化を効率化すること。

## ライトプラン

### 目的

まず購入しやすい入口商品にする。

### 内容

- スマホ対応命式ツール
- 生年月日・性別・出生時刻入力
- 年柱・月柱・日柱・時柱
- 蔵干
- 通変星
- 十二運
- 天中殺または空亡
- 五行バランス
- 大運
- 年運
- 単独HTML納品

### 含めないもの

- 鑑定文
- PDF鑑定書
- 管理画面
- 顧客保存
- 複雑な流派調整
- 独自チャート

### 価格帯メモ

初期は低めに出して実績を作る。

候補:

- 30,000円から50,000円

ただし、大運・年運まで含めるなら安売りしすぎない。

## スタンダードプラン

### 目的

占い師本人の鑑定スタイルに合わせる。

### 内容

ライトプランに加えて:

- 表示項目カスタマイズ
- 出生時刻不要設定
- 時柱非表示
- 柱順変更
- 表示名変更
- 色・デザイン調整
- 流派に合わせた軽微なルール調整
- WordPressや既存サイトへの設置相談

### 価格帯メモ

候補:

- 70,000円から120,000円

## プレミアムプラン

### 目的

鑑定書・独自メソッド・資料化まで含める。

### 内容

スタンダードに加えて:

- PDF鑑定書
- 独自チャート
- 行動キーワード表
- 鑑定文テンプレート
- 大運・年運コメント欄
- 先生独自の鑑定手順反映
- 顧客向け結果ページ

### 価格帯メモ

候補:

- 150,000円から300,000円

## オプション

- 鑑定文テンプレート追加
- PDF鑑定書
- 独自チャート
- 年運表追加
- 大運コメント追加
- 流派ルール調整
- デザイン調整
- WordPress埋め込み
- LINE誘導
- 予約導線
- AI鑑定文生成

## 受注前ヒアリング

必ず確認する項目:

- 現在使っている命式サイト・ソフト
- そのツールで困っている点
- 出生時刻を使うか
- 時柱を表示するか
- 通変星は日干基準でよいか
- 蔵干表に指定があるか
- 大運起算に指定があるか
- 年運は必要か
- スマホ中心か、PC中心か
- 鑑定文やPDFが必要か

## 見積もりルール

ライトプラン内で対応:

- 標準仕様の命式表示
- 大運・年運表示
- スマホ対応

追加見積もり:

- 独自流派の細かいルール
- 鑑定者独自メソッド
- PDF
- 管理画面
- 顧客保存
- 既存サイト組み込み
- AI生成

## 販売時の核

一番強い訴求はこれ。

> 無料サイトでは合わない、あなたの流派・鑑定スタイルに合わせた命式ツールを作ります。

次点:

> スマホで命式・大運・年運をすぐ確認できるので、鑑定準備や鑑定中の確認が楽になります。
