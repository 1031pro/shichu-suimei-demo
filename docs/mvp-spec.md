# MVP機能仕様

## 目的

ライトプランとして販売できる最小構成を定義する。

MVPでは、鑑定文やAI生成は扱わない。命式情報を正確に表示し、占い師がスマホで確認できる状態を作る。

## 入力項目

- 生年月日
- 出生時刻
- 性別
- 出生地
- 現在年

初期MVPでは出生地・真太陽時補正はオプション扱いにしてもよい。流派や鑑定者によって必要性が変わるため。

## 表示項目

基本命式:

- 年柱
- 月柱
- 日柱
- 時柱
- 天干
- 地支
- 蔵干
- 通変星
- 十二運
- 空亡

バランス:

- 木・火・土・金・水の五行バランス
- 陰陽バランス

運勢周期:

- 大運一覧
- 大運の開始年齢
- 大運の干支
- 大運の通変星
- 大運の十二運
- 現在の大運ハイライト
- 年運一覧
- 現在年の年運ハイライト

## 大運の扱い

大運はライトプランに含める。

理由:

- 実務鑑定では命式だけでは不足しやすい
- 「今どの運気か」を見るために必要
- 商品としての見栄えが大きく上がる
- 占い師向けツールとして最低限の実用性を持たせられる

ただし、大運ごとの鑑定文や詳細解釈はオプションとする。

## 画面構成

1. 入力フォーム
2. 命式表
3. 五行バランス
4. 大運
5. 年運
6. 補足情報

スマホ利用を前提に、縦スクロールで確認できる構成にする。

## 非対応にするもの

初期MVPでは以下を含めない。

- 鑑定文生成
- AI生成
- PDF出力
- 管理画面
- 顧客保存
- 複数流派切り替えUI
- 複雑な用神・喜忌判定
- 格局判定

これらは受注後のオプションまたは上位プランで扱う。
