# 節入りデータ方針

## 方針

新規アプリでは、節入りデータを外部APIへ毎回問い合わせない。

国立天文台 暦計算室の二十四節気・雑節 長期版から、節月切替に必要な12個の節を事前取得し、静的データとして同梱する。

## 出典

- 国立天文台 暦計算室 二十四節気・雑節 長期版
- URL: `https://eco.mtk.nao.ac.jp/cgi-bin/koyomi/cande/phenomena_sy.cgi`
- 標準時: `UT+9` 日本標準時

## 抽出対象

節月切替用に、二十四節気のうち以下の12個を使う。

- 小寒
- 立春
- 啓蟄
- 清明
- 立夏
- 芒種
- 小暑
- 立秋
- 白露
- 寒露
- 立冬
- 大雪

## 対応範囲

初期データは 1900 年から 2200 年まで。

既存制作物も同程度の範囲を持っていたため、通常の鑑定用途では十分。

## 生成スクリプト

```bash
node scripts/fetch-naoj-sekki.mjs 1900 2200
```

出力先:

- `data/setsuiri/setsuiri-1900-2200.json`
- `data/setsuiri/setsuiri-1900-2200.js`

## 運用

通常利用時は生成済みデータを読み込む。

更新が必要な場合のみ、生成スクリプトを再実行する。

これにより、通信エラー、外部サイト仕様変更、API停止によって鑑定ツールが動かなくなるリスクを避ける。
