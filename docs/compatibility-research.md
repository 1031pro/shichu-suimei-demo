# 相性診断（デモ）で扱う要素

生年月日だけで行う一般的な四柱推命の簡易相性として、以下を採用する。

1. **日干同士の五行関係** — 日干を本人の軸として、比和・相生・相剋を確認する。
2. **日干の干合** — 甲己、乙庚、丙辛、丁壬、戊癸の組み合わせを精神面での結び付きとして表示する。
3. **日支同士の関係** — 六合（親和）、冲（衝突・変化）、害（すれ違い）を確認する。日支は一般にパートナーシップや生活面の読み取りで重視される。
4. **五行バランスの補完** — 一方に少ない五行がもう一方で目立つかを、互いの持ち味として表示する。

このデモは「出生時刻なし」のため、時柱を必要とする詳細判断、用神・忌神、格局、蔵干を含む高度な合冲刑害の総合判定は対象外とする。また、一要素で良し悪しを断定せず、関係を整えるヒントとして表示する。

参考: [四柱推命による本当の相性の読み方](https://www.fourpillar.net/article/?id=2021090701)（日干・地支・喜神忌神の三点を総合する考え方）、[支合について](https://www.fourpillar.net/article/?id=2019121301)（地支の合）、[支合と干合の相性上の位置づけ](https://sup.andyou.jp/shicyusuimei/junishi/shigou/)。
