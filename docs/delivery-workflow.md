# 納品ワークフロー

四柱推命ツール制作を、ココナラで安定して受注・制作・納品するための流れ。

## 1. 事前相談

目的:

- 依頼者が何を作りたいかを確認する
- 標準プランで足りるか、追加オプションが必要かを判断する
- 流派差分があるかを早めに見つける

確認するもの:

- 既存の命式サンプル
- 鑑定書サンプル
- 表示したい項目
- 流派ルール
- スマホ利用の有無

## 2. 見積もり

標準ライトプラン:

- 命式表示
- 大運
- 年運
- スマホ対応
- PWA対応

追加見積もり:

- 独自ロジック
- 鑑定文
- PDF
- デザイン調整
- サーバー設置
- 顧客管理

## 3. 制作

手順:

1. 標準デモから顧客用プロジェクトを作る
2. アプリ名、色、表示項目を設定する
3. 流派プロファイルを設定する
4. サンプル生年月日で結果確認する
5. スマホ表示を確認する

## 4. 依頼者確認

依頼者に確認してもらう内容:

- 命式結果が合っているか
- 大運、年運が想定どおりか
- 表示項目に過不足がないか
- 文言に違和感がないか
- スマホで使いやすいか

## 5. 修正

修正の優先順位:

1. 計算ロジックの誤り
2. 表示項目の不足
3. スマホ表示の崩れ
4. 文言調整
5. 色や余白などの微調整

## 6. 納品

納品物:

- HTML/CSS/JS一式
- 節入りデータ
- 設定ファイル
- 簡易利用マニュアル

必要に応じて:

- サーバー設置
- ZIP納品
- 修正履歴
- 管理用メモ

## 7. 納品後オプション

追加提案:

- 鑑定文追加
- PDF鑑定書化
- 年運鑑定専用ページ
- 顧客管理機能
- デザインリニューアル
- 別流派バージョン作成
